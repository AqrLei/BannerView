Binaries
-----

```groovy

allprojects {
    repositories {
        maven { url 'https://jitpack.io' }
     }
}

dependencies {
        implementation 'com.github.AqrLei:BannerView:latestVersion'
}
```

Build
-----
To build:
```bush
$ git clone git@github.com:AqrLei/BannerView.git
$ cd BannerView/
$ ./gradlew.build
```
